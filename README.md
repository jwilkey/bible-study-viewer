# bible-study-viewer
A web application for viewing Bible studies written in the format described in https://github.com/jwilkey/bible-study

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Deploy using Github Pages
```
npm run deploy
```
